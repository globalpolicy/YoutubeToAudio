# YoutubeToAudio
A simple python 3.7 script to download audio from a youtube URL

```
usage: ytAudio.py [-h] [-s | -l] url

Retrieve audio files from youtube link

positional arguments:
  url          Youtube URL

optional arguments:
  -h, --help   show this help message and exit
  -s, --small  Small file
  -l, --large  Large file
  ```
